## Key Features

Our platform stands out with advanced features designed for modern media management needs.

<Tabs>

    <Callout kind="tip">Pro tip: Enable auto-backup to ensure your media assets are always protected.</Callout>

## Quick Start Guide

Get up and running in minutes with our simple setup process.

<Steps>
  <Step title="Create Account" icon="user">
    Sign up at kotakpintar.media and verify your email.
    
    <CodeGroup tabs="Web,API">
      ````javascript
      // Web registration
      const user = {
        email: 'user@example.com',
        password: 'securePass123'
      };
      fetch('/api/register', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(user)
      });
      ````
      ````bash
      # API registration
      curl -X POST https://api.kotakpintar.media/register \
        -H "Content-Type: application/json" \
        -d '{"email":"user@example.com","password":"securePass123"}'
      ````
    </CodeGroup>
  </Step>
  <Step title="Upload First Media" icon="upload">
    Navigate to the upload section and drag files or click to browse.
  </Step>
  <Step title="Organize Content" icon="folder">
    Create folders and add tags to keep your media organized.
  </Step>
</Steps>
