## Webhooks

Receive real-time notifications for integration automation.

<Expandable title="Webhook Configuration" default-open="true">
  Set up webhooks to trigger actions when media events occur.
  
  | Event | Description | Payload |
  |-------|-------------|---------|
  | media.uploaded | New file uploaded | File metadata and URL |
  | media.processed | Processing completed | Processing results |
  | media.deleted | Asset removed | Deletion confirmation |
  | folder.created | New folder added | Folder details |
</Expandable>

<CodeGroup tabs="Setup,Example">
  ````javascript
  // Webhook setup
  const webhookConfig = {
    url: 'https://your-app.com/webhook',
    events: ['media.uploaded', 'media.processed'],
    secret: 'your-webhook-secret'
  };
  
  fetch('/api/webhooks', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(webhookConfig)
  });
  ````
  ````json
  // Sample webhook payload
  {
    "event": "media.uploaded",
    "data": {
      "id": "abc123",
      "filename": "product.jpg",
      "url": "https://cdn.kotakpintar.media/abc123.jpg",
      "size": 2048576,
      "type": "image/jpeg"
    },
    "timestamp": "2024-01-15T10:30:00Z"
  }
  ````
</CodeGroup>

## Security Considerations

Maintain secure connections and protect your data.

- Use HTTPS for all webhook endpoints
- Implement authentication and authorization
- Validate webhook signatures to prevent spoofing
- Monitor integration health and error rates

<Callout kind="alert">Always test integrations in a staging environment before production deployment.</Callout>
