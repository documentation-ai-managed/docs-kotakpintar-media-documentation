## Tagging System

Implement a robust tagging strategy for enhanced asset discoverability.

<Expandable title="Tag Best Practices" default-open="true">
  - Use lowercase, hyphen-separated tags like "product-photos" or "marketing-videos"
  - Apply multiple relevant tags to each asset for comprehensive coverage
  - Create tag hierarchies for complex categorization needs
  - Regularly audit and clean up unused tags
</Expandable>

<Expandable title="Bulk Tagging" default-open="false">
  Apply tags to multiple assets simultaneously.
  
  <CodeGroup tabs="Web,API">
    ````javascript
    // Bulk tag via web interface
    const selectedAssets = ['asset1.jpg', 'asset2.mp4'];
    const tagsToAdd = ['marketing', 'q1-campaign'];
    
    fetch('/api/assets/bulk-tag', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${token}`
      },
      body: JSON.stringify({ assets: selectedAssets, tags: tagsToAdd })
    });
    ````
    ````bash
    # API bulk tagging
    curl -X POST https://api.kotakpintar.media/assets/bulk-tag \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer $TOKEN" \
      -d '{"assets":["asset1.jpg","asset2.mp4"],"tags":["marketing","q1-campaign"]}'
    ````
  </CodeGroup>
</Expandable>

## Maintenance Tasks

Regular maintenance ensures optimal library performance.

- Archive old assets to reduce active storage costs
- Delete duplicate files to free up space
- Update metadata for better organization
- Run integrity checks to verify file health

By implementing these management strategies, you'll maintain a well-organized, efficient media library that supports your content needs.
